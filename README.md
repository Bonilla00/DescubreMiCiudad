# DescubreMiCiudad
Proyecto académico - Plataforma de recomendación y ranking de lugares en la ciudad.

## Características Implementadas
- **Búsqueda de lugares:** Filtra por categorías y cercanía.
- **Favoritos:** Guarda tus lugares preferidos (localmente y en la nube).
- **Reseñas y Calificaciones:** Deja tus comentarios y estrellas, los cuales se mantienen guardados en la aplicación mediante persistencia local (`SharedPreferences`) y sincronización con el backend.
- **Geolocalización:** Encuentra lugares cerca de tu ubicación actual.
